# Prompt for GitHub Copilot: Quality Gate STANDARD Loop

## Instructions for the Copilot Agent

1. Execute the Quality Gate in STANDARD mode for the project.
2. If the Quality Gate STANDARD detects any errors or warnings, automatically fix them in the source code.
3. Rerun the Quality Gate STANDARD.
4. Repeat this process in a loop until the Quality Gate STANDARD passes with no errors or warnings.
5. When the process is complete, notify that the project passes the Quality Gate STANDARD with no issues.

## Notes
- Do not stop until the Quality Gate STANDARD is completely clean.
- Apply best practices for code quality and refactoring in each correction.
- Optionally, document the changes made in each iteration.

---

This prompt is designed for a LLM/Copilot agent to automate the process of passing the Quality Gate STANDARD for the project.
