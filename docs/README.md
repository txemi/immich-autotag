# Documentation

This folder contains all documentation for the project.

- **User documentation**: Guides, usage instructions, and information for end users are located here at the top level of `docs/`.
- **Developer documentation**: Internal resources, style guides, architecture, and technical details for contributors are in the `dev/` subfolder.

## Structure
- `dev/` — Developer and contributor documentation (see its README for details)
- Other files/folders — User-facing guides and documentation

If you are a developer or contributor, start with `dev/README.md`.
If you are an end user, browse the guides at this level.
